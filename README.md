# dotfiles

Install dotfiles

```console
git clone https://github.com/zhukovdm/dotfiles.git
cd dotfiles && ./install
```

Clone plugins into `~/.vim/pack/<github-user>/start/`

- [Dracula](https://draculatheme.com/vim) color theme
- [YouCompleteMe](https://github.com/ycm-core/YouCompleteMe) code-completion engine
